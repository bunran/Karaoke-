01
The Concept
What It Is
A Live Karaoke Entertainment Show
A professional, interactive live show where the hosts sing, battle, and invite the audience to participate — powered by real-time auto-tune and live mixing technology.
The Vision
Anyone Can Sound Great Live
Using built-in pitch correction and live vocal mixing, every performance sounds polished. The tech levels the playing field — the energy and personality are what the audience comes for.
The Difference
Real Tech + Real Entertainment
Unlike basic karaoke streams, KaraokeFlow uses a custom-built auto-tune mixer running live in the browser. Viewers experience a polished, professional sound on every stream.
The Culture
All Genres, All Languages
From Khmer R&B to English hip-hop, pop to ballads — KaraokeFlow celebrates music across cultures. Rooted in Southeast Asian diaspora culture with universal appeal.

03
Show Format
01
Host Performance
The hosts open each show with a song — setting the energy, testing the tech, and warming up the audience. Auto-tune on, mics hot, backing track loaded.
Opening Segment
02
Audience Sing-Along
Viewers request songs via live chat. Hosts play the backing track, display lyrics on screen, and invite the audience to sing along from home while watching the live stream.
Interactive
03
Guest Join-In
Selected viewers join the stream via duet or collab feature. They sing live with the hosts — their voice mixed in real time. Everyone gets the auto-tune treatment.
Live Collab
04
Karaoke Battle
Two audience members or guests go head-to-head singing the same song. Audience votes in real time via gifts, likes, or comments. Winner gets a shoutout and next song pick.
Competition
05
Challenge Mode
The hosts throw out a song challenge — hardest high note, funniest performance, most creative style. Clips get compiled into highlight reels for TikTok content after the stream.
Viral Content
06
CMG Feature Spot
Each show features one original CMG track — performed live with the full auto-tune mixer. This is the brand-building moment that connects KaraokeFlow to the CMG music catalog.
CMG Brand

06
Tech Stack
🌐
KaraokeFlow App
Custom-built browser app with live auto-tune, VU meters, pitch display, and mixer. Runs in Chrome — no install needed. Hosted free on GitHub Pages.
📹
OBS Studio
Free streaming software. Captures the browser audio output and webcam, combines them, and sends the stream to all platforms simultaneously.
🔊
Virtual Audio Cable
Routes the corrected voice audio from the browser mixer into OBS so the stream gets the tuned version — not the raw mic input.
🎧
Microphone + Headphones
A USB mic or XLR interface mic for clean vocal input. Headphones are essential to prevent audio feedback during live streams.
💻
Laptop / Desktop
Any Windows or Mac computer running Chrome can power the full setup. A dedicated streaming PC is ideal but not required to start.
🌍
GitHub Pages Hosting
The KaraokeFlow app is hosted completely free on GitHub Pages. HTTPS included — required for microphone access in the browser.

Launch Timeline
Phase 1 — Week 1-2
Setup & Testing
  • Host KaraokeFlow app on GitHub Pages
  • Set up OBS with virtual audio cable routing
  • Test mic input → auto-tune → stream output
  • Do 2-3 private test streams to verify audio quality
  • Set up TikTok, YouTube, and Instagram stream keys in OBS
Phase 2 — Week 3
Soft Launch
  • First public stream — announce 24 hours ahead on all platforms
  • Keep it simple: host songs + audience sing-along only
  • Test all 3 auto-tune modes live on air
  • Collect audience feedback on audio quality
  • Clip best moments for TikTok and Shorts
Phase 3 — Month 2
Full Format Launch
  • Introduce battle segments and guest join-ins
  • Set a consistent stream schedule (2-3x per week)
  • First CMG feature spot on air
  • Begin building clip content library from stream highlights
  • Enable TikTok gifts and YouTube Super Chats
Phase 4 — Month 3+
Growth & Monetization
  • Launch channel memberships and LIVE subscriptions
  • Pursue first brand sponsorship conversations
  • Expand song library — add Khmer catalog prominently
  • Build KaraokeFlow brand separately from CMG
  • Consider mobile-optimized version of the app
